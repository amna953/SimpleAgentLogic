# Что это?
Это простой ИИ-агент с поддержкой поиска в интернете (**Tavily**)
# Как его запустить?
1. Устанавливаем пайтон
2. Клонируем репозиторий ```git clone https://github.com/Intop1kDev/SimpleAgentLogic```
3. Устанавливаем зависимости (```pip install -r requirements.txt```)
4. Создаем **.env** файл и прописываем API-ключи для работы скрипта (OR_KEY, TAVILY)
5. Запускаем простой бекенд (```fastapi dev simple_backend.py```)
6. Открываем **127.0.0.1:8000/docs** (или вашу ссылку)
7. Шлем запрос на эндпоинт **/api/v1/chat** с текстом запроса
# Примечание
Лучше **НЕ** использовать данный скрипт в продакшене, бекенд не имеет авторизации, любой человек может использовать ваши API-ключи.
